Rust is a multi-paradigm programming language focused on performance and
safety, especially safe concurrency. Rust is syntactically similar to C++ and
provides memory safety without using garbage collection.

## Installation

`rustup` is an installer for the systems programming language Rust. Run the
following in your terminal, then follow the onscreen instructions:

```sh
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

### Essential tools

[Topgrade](https://github.com/r-darwish/topgrade) is a command line utility
used to keep your system update.

You can either use `cargo install` or use the compiled binaries from the
release page. The compiled binaries contain a self-upgrading feature.

```sh
cargo install topgrade cargo-update
```
