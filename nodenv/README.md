## About

Manage multiple NodeJS versions.

## Installation

This will get you going with the latest version of nodenv without needing a
systemwide install.

```sh
git clone https://github.com/nodenv/nodenv.git ~/.nodenv
cd ~/.nodenv && src/configure && make -C src
```

node-build is a command-line utility that makes it easy to install virtually
any version of Node, from source or precompiled binary.

```sh
mkdir -p ~/.nodenv/plugins
git clone https://github.com/nodenv/node-build.git ~/.nodenv/plugins/node-build
```
